Everything you will need to know about PlotJs

Getting Started

In order to use PlotJs you need to include it to your project

<script type="text/javascript" src="plotjs.min.js"></script>

To create a basic graph using PlotJs you only need to create the plane then plot the function

        var plane = new Plot({
            container: document.getElementById('plot')
        });

        plane.plot('sin(x)*5');

Note: You can either plot the function as a string sin(x)*5 or 3x^2-5x+3 or you can declare it explicitly using a javascript native function .
This function is equivalent the previously delared function

        plane.plot(function(x){
            return Math.sin(x)*10;
        });

You can be more creative and do this: 

        plane.plot(function(x){

           if(x < 0)
              return Math.cos(x)*5;
           else if(Math.round(x) == 0)
              return Math.pow(x,2);
           else
              return x;

        });

X is the variable of the X axes

Plane Configuration

This is are the plane default configuration you don't need to copy them each time you create a plane, you can change what ever you want to suit your needs

        var defaults = {

            container: '', //Required
            max: 21,    //Integer
            min: -21,  //Integer     
            strokeThickness: 3, //Integer 
            strokeColor: 'black',  //String
            lineColor1: "#AAAAAA", //String        
            lineColor2: "#DDDDDD",  //String           
            backgroundColor: 'white', //String
            zoom: true, //Boolean
            navigation: true, //Boolean
            height: '400px', //String
            width: '400px',  //String
            errorReporting: true, //Boolean
            foregroundColor: '#EEEEEE', //String
            guidelines: true,
            tickThickness: 2, //Integer
            tickColor: 'black', //String
            tickHeight: 16,  //Integer
            keyNavSpeed: 1, //Double
            fontSize: 10, //Integer (in px)
            fontColor: 'black',//String
            //Function, A callback function after a mouse displacement had occured
            dispCallback: function(disp){ return true;  },  
            //Function, A callback function after zoom
            zommCallback: function(){ return true;  }  

        }
  • Container: The container is the div that will contain the canvas of the plane
  • Max: The Maximum value of the plane
  • Min: The Minimum value of the plane
  • strokeThickness: The thickness of the line that devides the plane
  • strokeColor: The color of the stroke (RGBA or HEX or RGB)
  • lineColor1: The color of the guidelines of the plane
  • lineColor1: The color of the secondary guidelines of the plane
  • backgroundColor: The background color of the plane
  • Zoom: Zoomable or not (Boolean)
  • Navigation: Enable the navigation inside the plane
  • Height, Width: Specify the height and the width of plane
  • foregroundColor: The color of the foreground
  • guidelines: Use guidelines or not (it's recommended on low performance machines)
  • tickThickness: The thickness of the tick line
  • tickColor: The color of the tick line
  • tickHeight: The height of the tick in px
  • keyNavSpeed: The speed of the navigation using the keyboard
  • fontSize: The size of the font
  • fontColor: The color of the font
  • dispCallback(disp): Callback when you the navigate in the plane, it takes disp as parameter, disp is an object which contains disp.x and disp.y
  • zommCallback: The callback on zoom

Note: if the container is not set the plugin will not work and it will return an error !

  • equation: The equation of the function
  • resolution: The step that the loop take each time when it draws a new point, the high the value the better the results the lower the performance

    • Note: if the equation is not set the curve will not be drawn and the plugin will return an error !

    Note: there is two type of usage of the plot method the first one is plane.plot(theFunction) or with the configuration

              
          plane.plot('4x+3'); //The minimal usage with string as function

          plane.plot({

            borderColor: 'blue',
            equation: '4x+3'

          });   

          plane.plot({

                    //You can use the function as string or native function
            equation: function(x){ return 4*x+3; }, 
            borderColor: 'blue'

          })

    Other Methods

    Destroy

    To raze all the plane with its function use destroy

    Example 
              
          var plane = new Plot({
            container: document.getElementById('plot')
          }).plot('sin(x)*5');

          plane.destroy();

    Clear

    clear is used when you want to delete all the function and clearup the plane

    Example 
              
          var plane = new Plot({
            container: document.getElementById('plot')
          }).plot('sin(x)*5');

          plane.clear();

    destroyFunction

    destroyFunction is used to raze a function and keep the others

    Example 
              
          var plane = new Plot({
            container: document.getElementById('plot')
          });

          var fn = plane.plot('4x+3'),
              f2 = plane.plot('cos(1/x)');

          plane.destroyFunction(fn);
